# benchu-cli

![npm 版本](https://img.shields.io/badge/benchu-cli_v1.0.0-pink)

## 📖 简介

- 用于快速搭建 vue 或 react 项目的命令行工具

## 🚀 安装

```bash
npm install benchu-cli -g
```

## 📌 使用

### 1. 创建项目

```bash
benchu create
benchu create [project-name]
```

### 2. 查看脚手架版本

```bash
benchu -v
benchu -version
```

### 3. 更新脚手架到最新版本

```bash
benchu update
```
